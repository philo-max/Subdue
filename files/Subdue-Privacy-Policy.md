# Subdue — 隐私政策 (Privacy Policy)

**生效日期**：2026-06-19
**最近更新**：2026-06-19

欢迎使用 **Subdue (订阅刺客)**！我们非常重视您的隐私。本隐私政策旨在向您说明我们如何收集、使用、存储和保护您的个人信息。

---

## 1. 核心隐私承诺（不可妥协的红线）

1. **绝对零广告 (Zero Ads)**：无论现在还是未来，Subdue 永远不会接入任何广告 SDK 或第三方广告分析追踪工具。我们绝不通过转让或销售您的个人数据来谋利。
2. **本地存储优先 (Local-First)**：除非您主动选择商业版云同步服务，Subdue 开源版的所有订阅数据（包括服务名称、扣款金额、账单详情等）**默认 100% 仅保存在您的本地设备上**（PC 端保存在浏览器 IndexedDB，手机端保存在 AsyncStorage），除您之外的任何人（包括我们）都无法读取该数据。
3. **开源代码审计 (Open Source)**：我们的核心引擎、数据模型与计算逻辑全部在 GitHub 开源。任何人、开发者均可随时对代码进行安全与隐私审计。

---

## 2. 数据收集与使用说明

### 2.1 本地离线使用（开源版 / 免费版）
当您仅在本地使用本软件时：
* **数据收集**：我们**不收集**任何个人身份信息、设备指纹、位置数据或订阅记录。
* **汇率数据**：若您使用多币种功能，App 可能会通过外部 API（如汇率接口）获取最新的全球货币比率。在此过程中，您的订阅金额及敏感数据**绝不会**随该请求一同发送。

### 2.2 局域网同步 (LAN Sync)
当您使用 PC 与手机端的局域网配对同步功能时：
* 所有的配对和数据传输**纯粹在您的内网局域网（LAN）中直接点对点进行**。
* 数据完全不经过我们的服务器，没有任何外网泄露风险。

### 2.3 官方免费托管 Web 实例说明
如果您访问我们官方部署的免费 Web 实例，而不是自己部署：
* **运行日志**：我们仅保留网络运行所必需的最少服务器访问日志（如 IP 地址、请求时间，通常在 7 天内自动清理）。
* **业务数据**：即使使用官方实例，您的订阅账单明细依然存放在您本地浏览器的 IndexedDB 数据库中，不上传至服务器。

---

## 3. 数据掌控权

* **完全导出 (No Lock-in)**：您可以随时在系统设置中一键将所有账单数据导出为通用的 JSON 或 CSV 文件。
* **一键清除**：您可以在设置中随时点击“重置/清空所有数据”。一旦清空，保存在本地数据库中的数据将**永久且不可逆地被抹去**。

---

## 4. 政策修改

如果未来我们推出了需要云端备份的商业版，我们会显式向您请求授权，且只有在您主动开通并同意的情况下，才会进行云同步。任何对隐私政策的重大修改都将清晰地在 App 和 GitHub 仓库中进行披露。
