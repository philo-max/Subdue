**您是否也遇到过这些烦恼？**

**让 Subdue (订阅刺客) 来帮您彻底“驯服”自动扣费！**

Subdue 是一款**无广告、隐私优先、代码开源**的订阅管理与记账工具。我们坚持“本地优先 (Local-First)”原则，把所有数据掌控权原封不动地还给您。

## ✨ 核心亮点功能 (Key Features)

### 📊 1. 高颜值的订阅看板
告别简陋的表格！Subdue 拥有极具高级感的磨砂玻璃暗黑系视觉设计。
* **智能支出折算**：无论月付、周付还是年付，自动按您的主货币统一折算并汇总展示，清晰掌握每月开销。
* **公历扣款日历**：创新的日历网格热力图，标出每天需要扣费的订阅，今天扣了什么、明天将扣什么一目了然。

### 📸 2. 降低门槛：免打字智能录入
记账不需要繁琐的键盘输入！
* **截图本地识别**：直接把微信或支付宝的“自动扣费管理协议”页面截图，App 将在本地运行智能解析，自动提取服务名、金额和扣费周期并录入。
* **智能文本提取**：直接复制扣费短信、支付宝/微信账单文本，核心引擎一键识别参数并填充。

### 📡 3. 局域网安全加密同步 (LAN Sync)
* **无云对账**：没有云服务器，不上传敏感财务数据。只需通过 6 位安全配对码，PC 网页端和手机端即可在本地局域网内完成双向安全对账与合并。

### 🔔 4. 到期与涨价主动提醒
* **本地推送**：在服务自动扣款前 N 天，手机本地调度系统通知提醒您，及时考虑是否取消订阅。
* **涨价检测**：发现扣款金额与历史记录不符时，第一时间提醒，防止被悄悄“反向打折”。

### 💱 5. 实时多币种折算
* 支持美金、人民币、港币等多国主流货币，离线缓存最新汇率，自动为您转换汇总。

## 🔒 我们的三条硬红线 (Why Subdue?)

1. **绝对零广告**：没有任何横幅、开屏或插屏广告，给你最纯粹的使用体验。
2. **纯粹本地优先**：默认不联网收集任何个人数据，无需注册账号，所有账单留在您的手机和电脑里。
3. **数据完全自由**：我们永远不锁定您的数据。您随时可以一键导出完整的 JSON 和 CSV 文件进行备份或迁移。
